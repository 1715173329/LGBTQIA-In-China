關於 LGBTQIA

什麽是 LGBTQIA？

LGBTQIA 是壹個縮略詞，它代表女同性戀（Lesbian）、男同性戀（Gay）、雙性戀（Bisexual）、跨性別者（Transgender）、對自身性別認同產生疑惑的人/疑性戀（Questioning）、雙性人（Intersexual）、無性戀（Asexual），這類性少數群體需要受到尊重，因為我們生來如此。

這個項目為什麽建立？

由於中國的傳統文化背景，LGBTQIA 群體在中國非常不受待見，得不到他人的認可與尊重。建立這個項目是為了在開源工作者中找到 LGBTQIA 群體或是支持 LGBTQIA 平權的人們，從輿論上改善 LGBTQIA 群體在中國的狀況，盡管這會造成的影響並不大，但總比沒有好。

為什麽 LGBTQIA 需要平權？

LGBTQIA 是少數群體，盡管 LGBTQIA 也是普通人，但無法否認的是我們也與占多數的異性戀有所不同，政治權利無法受到完整的保障。自由、平等，在現代社會永遠都是最重要的，在中國，LGBTQIA 並沒有得到真正的自由平等，為了讓這類群體過上正常的生活，我們需要爭取平權。

如何尊重 LGBTQIA 群體？

就像妳尊重妳身邊的任何壹個朋友壹樣，把 LGBTQIA 們當做普通人。記住 LGBTQIA 不是任何形式上的壹種疾病，哪怕妳們有生理上的差異，LGBTQIA 也是正常人。同時，LGBTQIA 確實也需要比異性戀更多的關懷，因為 LGBTQIA 常常缺乏安全感，並且渴望被認同。



支持

如果妳支持 LGBTQIA 平權，歡迎在本項目 GitHub 倉庫的 SIGN.md 中留下妳的名字，並點壹個 Star 作為支持！同時，妳也可以寫壹篇關於中國 LGBTQIA 群體的文章，將原文地址和 Markdown 文件（作為備份）提供給我們，只要文章內容與 LGBTQIA 相關，或是 LGBTQIA 中的任意壹個群體相關，呼籲平權，我們都會通過。所有支持者的名字和他們提供的文章都可以在 GitHub 倉庫下的 SIGN.md 文件中看到。
